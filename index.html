About WURFL

WURFL is a continuously updated device description repository that utilizes sophisticated parsing logic and statistical analysis to properly handle false positives and incorrect information passed by the browser.

WURFL stands for Wireless Universal Resource FiLe. It "maps HTTP request headers to the profile of the HTTP client that issued the request", (wurfl.sourceforge.net).

Scientiamobile has written a publicly accessible page for testing browser and device capabilities using just a query string.
